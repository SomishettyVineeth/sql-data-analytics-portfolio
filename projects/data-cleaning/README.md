# SQL Data Cleaning Project

This project focuses on cleaning a real-world layoffs dataset using SQL.

## Tasks Performed
- Removed duplicate records
- Handled NULL values
- Standardized text fields
- Converted date formats

## Tools Used
- SQL
- Excel
- GitHub
