# Exploratory Data Analysis (EDA)

This project performs exploratory data analysis on a cleaned layoffs dataset
to understand trends, patterns, and business impact.

## Analysis Performed
- Total layoffs by company
- Layoffs by country
- Layoffs by industry
- Year-wise and monthly trends
- Identification of most affected sectors

## Tools Used
- SQL
- GitHub

## Dataset
This analysis uses the cleaned output from the data cleaning project.
